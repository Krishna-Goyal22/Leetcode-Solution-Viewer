My task was to create a website for searching leetcode solutions

FEATURES:-
1)You can search by question number or name
2)During search by question name,there is a autocomplete search box
3)After search if found, there are solutions in multiple programming languages(C++,Java,Python,etc)
4)There is a theme button for dark and light theme
5)The website is responsive across desktop and mobile devices
